Notfalloptionen bei Programmabsturz
===================================

Ebene: Spielzeug

Verhalten
---------

Sollte die Menüsoftware abstürzen, kann man diese Optionen verwenden, um den Hub schnell neuzustarten.

Die Notfalloptionen können aktiviert werden, indem der Bluetooth-Knopf für drei Sekunden gedrückt gehalten wird.
Dies wird durch einen Signalton und eine lilane Färbung der Bluetooth-LED bestätigt.

Nun kann, noch während der Bluetooth-Knopf gedrückt gehalten wird,
zwischen diesen Optionen gewählt werden:

* **Linker Knopf**: Neustart des Hubs
* **Mittlerer Knopf**: Starten der Menüsoftware
  Hinweis: Funktioniert nur, wenn die Menüsoftware abgestürzt ist.
* **Rechter Knopf**: Beenden von Spielzeug

Die Auswahl wird mit einem erneuten Signalton und eine grüne Färbung der Power-LED bestätigt.

In seltenen Fällen kann es sein, dass Spielzeug selber abgestürtzt ist. In diesem Fall kann auf dieses Menü
nicht zugegriffen werden und der Hub muss manuell neugestartet werden.

.. note::
    Inwiefern es Sinn macht, dieses Menü zu nutzen, gilt es noch zu evaluieren.
