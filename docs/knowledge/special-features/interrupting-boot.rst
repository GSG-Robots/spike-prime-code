Unterbrechung des Hochfahrprozesses
===================================

Ebene: Spielzeug & OS

Verhalten
---------

Laden der Menüsoftware verhindern
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Durch Gedrückthalten des linken und rechten Knopfes während des Starten kann verhindert werden,
dass die Menüsoftware startet. Das ist nützlich, wenn in der Menüsoftware oder in den Runs ein Fehler
ist, der den Hub zum Absturz bringt oder Code enthält, bei dem er sich aufhängt.

Laden von Spielzeug verhindern
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Durch Gedrückthalten des linken, rechten und des Bluetooth-Knopfes während des Starten kann verhindert werden,
dass Spielzeug startet. Das ist nützlich, wenn in Spielzeug ein Fehler
ist, der den Hub zum Absturz bringt oder Code enthält, bei dem er sich aufhängt.
